Accessibility Pad

A notepad application designed from the ground up with disability justice at its core — voice-powered, screen-reader ready, and fully navigable without a mouse.

Features

What It Does

  • Full screen reader support with ARIA-compliant markup throughout
  • High contrast mode and adjustable font sizes for low vision users
  • Complete keyboard navigation — no mouse required for any function
  • Voice-to-text input in four languages: English, French, Spanish, and Mandarin
  • Text-to-speech playback in English, French, Portuguese, and Mandarin
  • Hands-free operation mode for users with limited motor control
  • Zoom from 50% to 300% without layout breaking
  • Persistent notes with local storage — no account or login required
Case Study

Project Overview

Situation

Many note-taking apps fail to serve users with disabilities, language barriers, or alternative input preferences. Every feature here was designed with inclusion as the primary requirement from day one.

Tech Stack
Vanilla JavaScript Web Speech API Vosk-Browser CSS Custom Properties localStorage Semantic HTML5
Results
  • 100 / 100 Lighthouse Accessibility score
  • WCAG 2.1 Level AA verified — automated & manual testing
  • 100% keyboard accessible — no mouse required
  • <2s load time on 3G connection
  • Zero critical violations on axe DevTools scan
  • 4.5:1 contrast ratio maintained in dark mode
  • All core actions achievable in ≤ 3 keystrokes
